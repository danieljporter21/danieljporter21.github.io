Food Delivery App

React

Provided with a high fidelity wireframe to adhere to, we were tasked with building a React based food delivery app, utilising the Boostrap framework for the front end. Using a fetch to return JSON data from an API, we were able to display the reasturants & menus and then post an order back to the API.

My main focuses were ensuring the state of the jumbotron component updated to reflect the reasturant name that had been selected, the create order functionality by posting back to the API, as well as the modal popup and progress bar that appeared when placing the order

Note; if you are cloning this project from Github, follow the additional instructions for installing the API locally on your machine!
