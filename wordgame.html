Guess the Word Game

Javascript

The first group project I was involved in. Using a fetch from JSON data, we had to generate a randomised guess the word game with Javascript, with a score and timer to make things more interesting.

My focuses for this project included the click event for the 'Start Game' button, creating the scoring system and recording the number of guesses remaining
