Collection App

PHP & SQL Databases

This was the second solo project I worked on at iO-academy. Building on the HTML & CSS I had learnt, I was tasked with creating a PHP based collection application that required displaying items on a webpage from a MySQL database, featuring an option to add the database, with some basic santisation and validation, as well as a first foray into unit testing.
