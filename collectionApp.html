Collection App

PHP & SQL Databases

This was the second solo project I worked on at iO-academy. I chose to create a collection of Marvel films that I own. Using PHP, I accessed a relational MySQL database that was made up from several tables with one to many relationships and displayed the items on a page. I implemented a form to add to the collection, with the inputs sanitised and validated to protect against SQL injection. All possible functions were unit tested.
